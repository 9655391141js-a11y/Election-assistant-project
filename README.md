🗳️ Smart Election Assistant

📌 Project Overview

Smart Election Assistant is an interactive web application designed to simplify and explain the election process for users. It helps especially first-time voters understand voting in a simple, engaging, and user-friendly way.

---

🚀 Features

- 📅 Election Timeline
  Step-by-step explanation of the election process.

- 🧠 Flashcards
  Learn important election terms like ECI, EVM, VVPAT, NOTA.

- ❓ Quiz Section
  Test your knowledge with multiple-choice questions.

- 🤖 Chat Assistant
  Ask questions and get instant answers about voting.

---

💡 Purpose

- Increase voter awareness
- Educate users about elections
- Provide easy and interactive learning

---

🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Google Antigravity (AI-powered prompt-based development)
